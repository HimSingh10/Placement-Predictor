# 🎓 Engineering Placement Predictor

An end-to-end Machine Learning project that predicts student placement using academic and skill-based features.

## Features
- XGBoost Model with Hyperparameter Tuning
- SMOTE for handling class imbalance
- Feature Importance Visualization
- Confusion Matrix & Metrics
- Interactive UI using Streamlit

## Tech Stack
- Python
- Scikit-learn
- XGBoost
- Streamlit
- Pandas, NumPy

## Model Performance
- Optimized using F1-score
- Balanced predictions with threshold tuning

## Run Locally
```bash
pip install -r requirements.txt
python train_pipeline.py
streamlit run app.py
